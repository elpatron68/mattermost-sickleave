### Changed

- Plugin ID namespace to `de.medisoftware.mattermost-sickleave`

## [0.1.0] - 2026-06-30

### Added

- Sick leave slash commands, channel header menu, and webapp date-picker modals
- HR channel posts with thread replies for updates, extensions, and case closure
- Configurable report hashtag and slash command trigger
- English and German UI (informal *Du* in DE)
- Unit tests and Linux release bundles via `make dist`
